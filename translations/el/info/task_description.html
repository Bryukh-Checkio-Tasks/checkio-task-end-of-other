Για να εξασκηθούν στη γλώσσα τα ρομπότ μας θέλουν να μάθουν για τις καταλήξεις.

Σε αυτή την αποστολή, σου δίνεται ένα σύνολο (set) από λέξεις γραμμένες με πεζά γράμματα. Σκοπός είναι να εξετάσεις αν υπάρχει ένα ζευγάρι λέξεων, τέτοιο που η μια λέξη να είναι το τέλος της άλλης (να είναι η κατάληξη της άλλης). Για παράδειγμα: {"hi", "hello", "lo"} -- "lo" είναι το τέλος της λέξης "hello", επομένως το αποτέλεσμα είναι Αληθές (True).

Είσοδος: Λέξεις σε μορφή συνόλου strings.

Έξοδος: Η τιμή Αληθής (True) ή Ψευδής (False).

Παράδειγμα: 

checkio({"hello", "lo", "he"}) == True
checkio({"hello", "la", "hellow", "cow"}) == False
checkio({"walk", "duckwalk"}) == True
checkio({"one"}) == False
checkio({"helicopter", "li", "he"}) == False

Πώς χρησιμοποιείται: Εδώ θα μάθεις να διασχίζεις τον τύπο δεδομένων σύνολο (set) και να χρησιμοποιείς συναρτήσεις για τον τύπο δεδομένων string.

Προϋποθέσεις: 2 ≤ len(words) < 30
all(re.match(r"\A[a-z]{1,99}\Z", w) for w in words)
