Для практики в лінгвістиці Роботи хочуть вивчити суфікси.

У цьому завданні заданий набір слів в нижньому регістрі. Перевірте чи є в цьому наборі пара слів, така, що одне слово закінчується іншим (суфікс чи збіг).
Наприклад: {"hi", "hello", "lo"} - "lo" це закінчення "hello", так що результатом буде True.

Зауваження: Для цього завдання ви можете {% if "python 2" in interpreter|lower %} прочитати про тип даних set і про функції роботи з рядками символів. {% else %} прочитати про тип даних set і про функції роботи з рядками символів. {% endif %}

Вхідні дані: Слова як набір (set) рядків (str).

Вихідні дані: True або False, як логічний вираз.

Приклади: 

checkio({"hello", "lo", "he"}) == True
checkio({"hello", "la", "hellow", "cow"}) == False
checkio({"walk", "duckwalk"}) == True
checkio({"one"}) == False
checkio({"helicopter", "li", "he"}) == False

Як це використовується: У цьому завданні ви познайомитеся з тим, як ітерувати тип даних set і також з деякими корисними функціями.

Передумови:
2 ≤ len(words) < 30
