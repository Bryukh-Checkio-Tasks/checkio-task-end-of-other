Per l'allenamento linguistico i Robot vogliono imparare di più sui suffissi.

In questo compito ti viene dato un insieme di parole in carattei minuscoli. Controlla se c'è una coppia di parole tale che una sia la parte finale dell'altra (ovvero un suffisso). Ad esempio: {"hi", "hello", "lo"} -- "lo" è la parte finale di "hello", dunque il risultato è True.

Input: Parole come insieme di stringhe.

Output: Vero o falso, come booleano.

Esempio: 

checkio({"hello", "lo", "he"}) == True
checkio({"hello", "la", "hellow", "cow"}) == False
checkio({"walk", "duckwalk"}) == True
checkio({"one"}) == False
checkio({"helicopter", "li", "he"}) == False

Come si usa: Imparerai ad iterare su insiemi e stringhe.

Prerequisiti: 2 ≤ len(words) < 30
all(re.match(r"\A[a-z]{1,99}\Z", w) for w in words)
